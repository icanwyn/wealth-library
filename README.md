# Wealth 財 — Digital Library

A Japanese-aesthetic digital library of the **top 100 works of financial literature**.

## Features

- **Dynamic hero** — sakura particles, seigaiha waves, mist mountains, parallax
- **Bookshelves** — 100 spines with hover tilt + short description tooltips
- **Book detail** — essence (神髄), top 10 meat paragraphs, expandable real-world applications
- **Search & category filters**
- **Washi / sumi / vermillion / gold** visual system

## Local preview

```bash
cd ~/Desktop/Wealth
npx serve
# or open index.html in a browser
```

## Stack

Static HTML, CSS, and vanilla JavaScript. Deployable on Vercel with zero build step.
